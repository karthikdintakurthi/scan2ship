üìã Integration Instructions

How to complete the integration:

  1. Generate API Key: Click "Generate API Key" above to create your unique API key
  2. Configure Shopify App: In your Shopify Partners dashboard, set the webhook URL to: https://your-app.vercel.app/api/shopify/webhooks
  3. Connect Store: Enter your shop domain and click "Connect Shopify Store"
  4. Authorize: You'll be redirected to Shopify to authorize the connection
  5. Test Integration: Create a test order in Shopify to verify the sync works

üîß Webhook Configuration

Configure these webhooks in your Shopify app settings:

  • Order Creation: https://your-app.vercel.app/api/shopify/webhooks (Topic: orders/create)
  • Order Updates: https://your-app.vercel.app/api/shopify/webhooks (Topic: orders/updated)
  • Order Payment: https://your-app.vercel.app/api/shopify/webhooks (Topic: orders/paid)
  • Order Cancellation: https://your-app.vercel.app/api/shopify/webhooks (Topic: orders/cancelled)
